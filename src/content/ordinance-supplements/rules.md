## 2. 主要パラメータ表
各パラメータを「厳格案」「標準案」「調整案」の三段階で提示。導入自治体は地域特性に応じて選択する。

| パラメータ | 厳格案 | 標準案 | 調整案 | 根拠・留意事項（※導入時に出典URL・確認日を付記のこと） |
| :--- | :--- | :--- | :--- | :--- |
| **廃棄費用積立金** | 2万円/kW | 1.5万円/kW | 0.8万円/kW | 撤去費用中央値約1.6万円/kW（資源エネルギー庁）。国制度との差額設計 |
| **調整池設計降雨** | 50年確率 | 30年確率 | 10年確率 | 林野庁中間とりまとめの選択肢内 |
| **流出係数上限** | 0.7 | 0.8 | 0.95 | 水源涵養目的を明示すれば合理性を説明し得る |
| **傾斜度制限（民有林）** | 15° | 20° | 25° | 例外許可条項を残せば適法性を高める設計 |
| **傾斜度制限（一般）** | 20° | 25° | 30° | ただし書で例外を許容すれば比例原則を充足し得る |
| **住居離隔距離** | 200m | 150m | 50m | 条件付きなら過大とは言い難い。短縮申請制度を併設 |
| **通学路離隔距離** | 100m | 80m | 30m | 児童安全目的。批判されにくい |
| **河川離隔距離** | 50m | 30m | 20m | 河川法施行令と整合 |
| **造成面積閾値** | 1,000㎡ | 3,000㎡ | 5,000㎡ | 小規模免除（50kW未満）との整合 |
| **廃止届出事前** | 180日前 | 120日前 | 60日前 | 水源復元計画の審査を含む |
| **事業承継届出** | 21日以内 | 30日以内 | 60日以内 | 承継後の管理責任を明確化する期間 |
| **広葉樹活着率** | 80% | 70% | 60% | 林野庁造林基準の「良好」判定を参照 |
| **管理不全判定** | 6ヶ月 | 1年 | 2年 | パネル特性で正当化可能 |
| **環境保全協力金** | 500円/㎡ | 300円/㎡ | 50円/㎡ | **努力義務（任意）。許認可判断とは一切連動しない。使途・執行状況を年次公表** |
| **過料（秩序罰）** | 5万円 | 5万円 | 5万円 | 地方自治法上限。行政上の秩序罰であり刑事罰ではない |
| **罰金（命令違反）** | 30万円以下 | 20万円以下 | 10万円以下 | **刑事罰。検察庁との事前協議が必要。** 過料とは法的性質が異なる |
| **広葉樹植栽倍率** | 2.0倍 | 1.5倍 | 1.0倍 | 水源涵養目的であれば合理性を説明し得る |

## 4. 届出情報の取得範囲（第9条の2関連）

> **⚠️ 取得範囲は個人情報保護担当・法務担当と協議の上、通常の不動産取引・行政手続で取得済みの範囲に限定すること。**

一　不動産登記事項証明書に記載された事項
二　売買契約書又は媒介契約書に記載された買主又は委託者に関する事項
三　宅地建物取引業法第35条の規定による重要事項説明書に記載された事項
四　宅地建物取引業者が取得した顧客情報のうち、買主の氏名又は名称及び住所に関する事項
五　農地法第3条又は第5条の許可申請書に記載された事項（農地の場合に限る。）
